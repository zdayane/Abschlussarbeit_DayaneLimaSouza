# Bachelorarbeit

### „Prototypische Umsetzung des Workflows einer webbasierten Nachrichtenplattform mit Fokus auf unabhängigem Journalismus“

## Ziel der Arbeit

Das Ziel dieser Arbeit besteht darin, einen Prototypen für eine webbasierte Nachrichtenplattform zu entwickeln, der den Workflow des unabhängigen Journalismus unterstützt und optimiert.

Der Fokus liegt darauf, eine Plattform bereitzustellen, die Journalisten und Redakteuren das Erstellen, Bearbeiten und Veröffentlichen von Inhalten erleichtert, um effizienter arbeiten zu können.

Der Prototyp soll diese Anforderungen erfüllen und damit den Arbeitsablauf von Journalisten und Redakteuren verbessern.
